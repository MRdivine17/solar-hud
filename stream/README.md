# Stream Files for Custom Minimap

Place your custom minimap files here:

1. **minimap_layout.ytd** - Custom minimap texture file
2. **minimap.gfx** - Custom minimap graphics file

These files will replace the default GTA V minimap appearance.

## How to add your files:

1. Copy `minimap_layout.ytd` to this folder
2. Copy `minimap.gfx` to this folder
3. Restart the resource

The files will be automatically streamed to clients.
